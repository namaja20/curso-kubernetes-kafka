# Entorno de prácticas (apoyo)

En esta carpeta se incluyen, cuando el diseño del curso lo requiera, scripts, manifiestos auxiliares o definiciones (por ejemplo entornos con **Kind**, **k3d** o **Docker Compose**) pensados para ejecutarse desde el **terminal del GitHub Codespace** vinculado al repositorio.

El docente indicará qué comandos utilizar y en qué orden, según la configuración del aula.
