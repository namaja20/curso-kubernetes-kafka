Material de soporte para la parte teórica del bloque 1 (antes de los laboratorios 1–4).
Cada tema está pensado para una explicación de **20–40 minutos** y se encadena con el siguiente; pueden impartirse seguidos como una sola sesión introductoria de aprox. **2,5 h**.

| Orden | Tema | Archivo | Tiempo orientativo |

| 1 | Contenedores vs VMs y rol de Docker | [`01-contenedores-vs-vms.md`](01-contenedores-vs-vms.md) | 25–30 min |

## Cómo usar este material

- Cada archivo arranca con la **idea clave en 30 segundos**: úsala como diapositiva inicial o como cierre cuando preguntes “¿qué se llevan de aquí?”.
- Los **diagramas Mermaid** se renderizan directamente en GitHub; también pueden exportarse para presentación.
- La sección **errores típicos / preguntas frecuentes** anticipa las dudas que suelen aparecer en aula: tenerlas presentes evita perder tiempo improvisando respuestas.
- El **puente al siguiente tema** muestra qué hilo recoger para encadenar la siguiente parte sin que el grupo pierda el contexto.
